# 🎬 คลังหนังของฉัน · My Movies in iTunes

เว็บเก็บข้อมูลหนังที่ซื้อไว้ใน **iTunes / Apple TV** — จัดการคอลเลกชันส่วนตัวแบบละเอียด
(หนังที่แค่ดูแต่ไม่ได้ซื้อ จะไม่อยู่ในนี้)

> Developed by **สิรวิชญ์ เผ่าผา** · ข้อมูลหนังจาก TMDB · OMDb

| หน้า | ทำอะไร |
|---|---|

## 🚀 วิธีรัน (local)

ไม่ต้อง build อะไร — เป็นเว็บไฟล์ธรรมดา แค่เปิดผ่านเซิร์ฟเวอร์เล็กๆ:

```bash
# วิธีที่ 1 — Python
python -m http.server 8770

# วิธีที่ 2 — Node
npx serve -l 8770
```

แล้วเปิด `http://localhost:8770`

## 🧱 เทคโนโลยี

- **React 18** (โหลดจาก CDN — ไม่มีขั้นตอน build)
- **ตัวแปลเทมเพลตของเราเอง** (`dc-runtime.js`) — แปลงเทมเพลต HTML ให้เป็น React
- เก็บข้อมูลใน **localStorage** (จะย้ายขึ้นคลาวด์ในเวอร์ชันถัดไป)
- โปสเตอร์/ข้อมูลหนังจาก **TMDB** + **OMDb**

## 📁 โครงสร้างไฟล์

```
index.html       หน้าหลัก (โหลด React + ไฟล์ของเรา)
app-core.js      สมองแอป — state + ค่าคงที่ + ข้อมูล + เวอร์ชัน + lifecycle
app-actions.js   การกระทำ/ปุ่มทั้งหมด (ต่อเข้า App.prototype)
app-render.js    สร้างหน้าจอ (renderVals) + จุดเริ่มแอป
template.html    หน้าตา (markup ของทั้งแอป)
dc-runtime.js    ตัวแปลเทมเพลต → React
cloud.js         เชื่อม Supabase (sync ข้อมูลคลาวด์)
api.js           ค้นหา + ดึงข้อมูลหนังจริง (TMDb/OMDb)
styles.css       สไตล์ฐาน + animation
```
